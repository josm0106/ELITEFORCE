제33단원

THA 재치환술 (Revision of THA)

1. 고관절 재치환술 적응증

• 조기증상: 통증 또는 관절 운동성 소실

• 기계적 합병증: 파열, 이완, 돌출

• 생물학적 합병증: 골용해, 골용해성 부위 파괴

• 감염: 초기감염 또는 원발성감염

• 기타: 나선형 또는 이단계 절차, 뼈 소실 재구성

2. 재치환 전 평가

• 임상적 증상 평가 (pain, function, ROM)

• 방사선학적 검사: X선, CT - 뼈 결손 정도, 임플란트 위치

• 혈액검사: 감염 지표 (ESR, CRP)

3. AAOS 뼈 결손 분류

• Type I: 분할된 결손

• Type II: 대공 결손

• Type III: 복합 결손

4. 재치환 수술 기법

4-1. 이전 임플란트 제거

- 망치와 끌

- 고속 버 (high-speed burr)

- 초음파 장치

4-2. 뼈 결손 재건

- 소형 결손 (<2cm): 시멘트만

- 중형 결손 (2-4cm): 골이식 + 시멘트

- 대형 결손 (>4cm): 구조적 골이식 + 금속 추가

5. 합병증

• Infection: 감염

• Re-dislocation: 재탈구

• Aseptic loosening: 무균 이완

복습 문제

단답형

1. THA 재치환술의 주요 적응증 3가지를 나열하시오.

답: Aseptic loosening, Infection, Mechanical failure

2. AAOS 뼈 결손 분류 Type III는 어떤 결손인가?

답: 복합 결손 (Combined deficiency)

3. 대형 뼈 결손 재건 시 사용하는 방법은?

답: 구조적 골이식과 금속 추가 부위의 조합 사용

5지선다형

1. THA 재치환술에서 가장 흔한 적응증은?

① 감염

② 무균 이완

③ 베어링 표면 마모

④ 컴포넌트 오정렬

⑤ 불안정성

정답: ②

해설: 무균 이완(aseptic loosening)이 THA 재치환술의 가장 흔한 적응증입니다. 장기간 사용으로 인한 진행성 골용해로 인해 임플란트가 이완됩니다.

2. 고속 버로 임플란트 제거 시 주의할 점은?

① 시간이 오래 걸림

② 비용이 많이 듦

③ 열로 인한 골손상과 파편

④ 소음 문제

⑤ 마모가 빠름

정답: ③

해설: 고속 버 사용 시 생성되는 열은 골괴사를 초래하고, 임플란트 파편이 산재될 수 있습니다.

3. 중형 뼈 결손(2-4cm)의 재건 방법은?

① 시멘트만 사용

② 골이식 단독

③ 골이식과 시멘트 조합

④ 금속 보강물

⑤ 이식편만 사용

정답: ③

해설: 중형 결손은 구조적 골이식과 시멘트를 함께 사용하여 적절한 지지력을 제공합니다.

4. 소형 뼈 결손(<2cm)의 처리 방법은?

① 골이식 필수

② 시멘트만 사용

③ 금속 임플란트

④ 흉부 이식

⑤ 보수적 치료

정답: ②

해설: 소형 결손의 경우 시멘트만으로도 충분한 지지력을 제공할 수 있습니다.

5. THA 재치환 후 재탈구 위험을 감소시키는 방법은?

① 더 큰 임플란트 사용

② 최적의 컴포넌트 정렬

③ 항상 시멘트 사용

④ 완전한 고정

⑤ 재활 최소화

정답: ②

해설: 최적의 컴포넌트 정렬과 배치가 재탈구 위험을 효과적으로 감소시킵니다.
