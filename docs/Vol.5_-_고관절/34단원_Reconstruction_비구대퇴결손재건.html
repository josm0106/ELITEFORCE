제34단원

비구 및 대퇴 결손의 재건 (Reconstruction of Acetabular & Femoral Deficiencies)

1. 비구 결손의 분류 및 재건

1-1. AAOS 비구 결손 분류

• Type I: 분할된 결손 (segmental deficiency)

• Type II: 대공 결손 (cavitary deficiency)

• Type III: 복합 결손 (combined deficiency)

• Type IV: 골반 불연속 (pelvic discontinuity)

• Type V: 관절증 (arthrodesis)

1-2. 비구 결손의 재건 방법

- 소형 결손: 시멘트로 충전

- 중형 결손: 골이식 + 시멘트

- 대형 결손: 구조적 골이식 + 금속 보강물

- 골반 불연속: 구조적 골이식 + 장골 나선형 고정

1-3. 세분화된 비구 재건 기법

분할된 결손:

- 비구의 중앙 지지 손실을 특징으로 함

- 지정된 부위의 재건술 필요

대공 결손:

- 비구 내강의 국소적 결손

- 대부분의 경우 구조적 골이식 불필요

복합 결손:

- 우월부 및 중앙부 대공 결손

- 상대적 수술적 계획 필요

구조적 골이식:

- 골판 구조 사용 (femoral head, 동종골)

- 생역학적 적절한 함유, 견고한 고정

2. 대퇴 결손의 분류 및 재건

2-1. AAOS 대퇴 결손 분류

• Type I: 분할된 결손 (segmental deficiency)

• Type II: 대공 결손 (cavitary deficiency)

• Type III: 복합 결손 (combined deficiency)

• Type IV: 골절 (fracture)

• Type V: 대퇴 협착 (femoral stenosis)

• Type VI: 대퇴 불연속 (femoral discontinuity)

2-2. 대퇴 재건 방법

분할된 결손:

- 피질골 및 해면골 손실

- 전체 미소적 강화 및 구조적 지지 필요

대공 결손:

- 국소적인 뼈 소실

- 시멘트 또는 비시멘트 재건

복합 결손:

- 우월부 및 내측부 결손

- 더 광범위한 골이식 필요

3. 고급 재건 기법

3-1. 구조적 골이식의 선택

• 동종 대퇴두 (femoral head)

• 동종 장골 (iliac crest)

• 동종 피질골

3-2. 금속 보강물의 사용

• 대형 후방 또는 우월부 분할된 결손

• 권장되는 경우: 3cm 이상의 상방 변위

• 기술: 대공 및 구조적 골이식과 함께 사용

3-3. 다양한 기술

- 골반 절단을 포함한 압축 오정렬 기법

- 흉부 지지 제공 및 생물학적 중심

- 해부학적 중심점 회복

4. 대퇴 골절 관련 합병증

• 대퇴 골절, 피질 침투 및 추가 대퇴골 파괴

• 주변부 정맥 혈관구조 근처의 기술적 어려움

• 대퇴 관주위 조직의 방사선학적 추적

5. 골반 불연속 관리

• 드물지만 복잡한 합병증

• 극도의 골반 불안정성으로 이어질 수 있음

• 최소 5년 추적-관찰 필요

• 다양한 비구 재보강 기법 및 탈륨 교체가 필요

6. Della Valle & Paprosky 분류

6-1. Type I

• 최소한의 뼈 손실, 무손상 대퇴증

• 대부분 Type I 대퇴는 시멘트 또는 비시멘트 일차 길이 스템으로 재건

6-2. Type II

• 광범위한 골용해성 대공 뼈 손실

• 하부 레벨까지의 대공 손실

• 일반적으로 추가 내시경 촉진이 대퇴 복구를 고려하는 상황에서 8인치 또는 더 이상 권장됨

6-3. Type III

• 광범위한 골용해성 대공 뼈 손실

• 대퇴 협착뿐만 아니라 불완전한 뼈 손상을 초래함

• 대부분의 경우 Calcar 대체 또는 모듈식 대퇴 교체가 필요함

• 더 광범위한 대공 또는 구조적 뼈이식

6-4. Type IV

• 대퇴 골절로 이어진 광범위한 내측 부작용 불충분한 안정성

• 구조적 골이식 및 모듈식 대퇴 교체 필요

6-5. Type V

• 불완전한 치유 및 불충분한 뼈 질량

복습 문제

단답형

1. AAOS 비구 결손 분류를 나열하시오.

답: Type I (segmental), Type II (cavitary), Type III (combined), Type IV (pelvic discontinuity), Type V (arthrodesis)

2. 대형 비구 결손의 재건 방법은?

답: 구조적 골이식과 금속 보강물(metal augments) 조합 사용

3. Della Valle 분류 Type II의 특징은?

답: 광범위한 골용해성 대공 뼈 손실로 하부 레벨까지 진행

5지선다형

1. AAOS 비구 결손 분류에서 가장 복잡한 유형은?

① 분할된 결손

② 대공 결손

③ 복합 결손

④ 골반 불연속

⑤ 관절증

정답: ④

해설: 골반 불연속(Type IV)은 비구와 골반의 연속성이 소실된 가장 복잡한 유형으로, 구조적 골이식과 장골 나선형 고정이 필요합니다.

2. 대퇴두 골이식이 주로 사용되는 비구 결손은?

① 소형 결손

② 분할된 우월부 결손

③ 대공 후방 결손

④ 골반 불연속

⑤ 관절증

정답: ②

해설: 동종 대퇴두는 구조적 골이식으로 사용되며, 특히 분할된 우월부 결손의 재건에 효과적입니다.

3. 금속 추가 부위가 권장되는 경우는?

① <1cm 우월부 변위

② 1-2cm 우월부 변위

③ 2-3cm 우월부 변위

④ >3cm 우월부 변위

⑤ 모든 경우

정답: ④

해설: 일반적으로 3cm 이상의 우월부 변위가 있을 때 금속 보강물의 사용이 권장됩니다.

4. Della Valle Type IV 대퇴 결손의 특징은?

① 최소한의 뼈 손실

② 국소 대공 손실

③ 광범위한 골용해 손실

④ 골절로 인한 광범위한 손실

⑤ 협착만 있음

정답: ④

해설: Type IV는 골절로 이어진 광범위한 내측 부작용으로 인한 불안정성을 특징으로 하며, 구조적 골이식과 모듈식 교체가 필요합니다.

5. 골반 불연속 관리의 최소 추적 기간은?

① 1년

② 2년

③ 3년

④ 5년

⑤ 10년

정답: ④

해설: 골반 불연속은 드물지만 복잡한 합병증으로, 최소 5년의 추적-관찰이 필요합니다.
