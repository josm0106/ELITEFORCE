제22단원

Supracondylar fracture of humerus

상완골 과상 골절

교과서 내용 정리

1) 상완골 과상 골절 (Supracondylar Fx of humerus)

(1) 정의 및 빈도

• 상완골 원위부 골절 중 가장 흔한 골절

• 대부분 신전형(Extension type)으로 발생

• 5-8세 아동에서 높은 빈도

(2) 분류

#### Gartland 분류 (Modified)

• Type I: 미위치 골절(Undisplaced fracture)

• 후방 피질골이 정상 위치에 있음

• 보존적 치료 가능

• Type II: 부분 전위 골절(Displaced fracture, intact periosteum)

• 후방 피질골이 부분적으로 손상

• 통상적 폐쇄 정복 가능

• Type III: 완전 전위 골절(Complete displacement)

• 완전히 전위되어 주변부와 연결 없음

• 대부분 수술적 치료 필요

• Type IV: 회전 전위(Rotational displacement)

• 나선형 전위

• 해부학적 정복 어려움

#### 신전형 vs 굴곡형

• 신전형(Extension type, ~95%)

• 손바닥 부위에 넘어지며 발생

• 근위편이 전방으로 편위

• 굴곡형(Flexion type, ~5%)

• 팔꿈치에 직접 외상

• 근위편이 후방으로 편위

(3) 임상 증상 및 검진

• 병력: 넘어지면서 팔꿈치에 외상(fall on outstretched hand, FOOSH)

• 주증상

① 팔꿈치 부종 및 동통

② 팔꿈치 운동 제한

③ 팔꿈치 변형(각변형)

• 중요한 검진

① 신경학적 검진: 정중신경, 요골신경, 척골신경 기능 평가

② 혈관 검진: 요골 동맥 박동, 손의 색깔, 온도, 모세혈관 재충전

③ 피부 상태 확인: 개방성 골절 여부

④ 구획증후군(Compartment syndrome) 가능성 평가

(4) 신경 손상 (Nerve Injury)

#### 신경 손상 빈도 및 특징

• 전체 supracondylar 골절의 10-15% 정도

• Extension type에서 더 흔함

• 신경 손상의 순서

① 정중신경(Median nerve) - 가장 흔함

② 요골신경(Radial nerve)

③ 척골신경(Ulnar nerve)

④ 요골 신경의 후골간(Posterior interosseous nerve, PIN)

• 폐쇄 정복 후 신경 기능 회복이 대부분 일어남

• 신경염전(Entrapment)이나 신경절단(Neurotmesis)도 가능

(5) 혈관 손상 (Vascular Injury)

• 비교적 드물지만 심각한 합병증

• 대부분 폐쇄 정복으로 복구 가능

• 보존적 대기 요법(Conservative wait and see approach) 가능

• 손의 색깔 개선, 맥박 회복 확인 중요

(6) 영상 검사

#### 초기 평가

• 정면 및 측면 X선: 기본 검사

• Posterior fat pad sign: supracondylar 골절의 간접 증거

• Anterior humeral line: 정복 상태 평가

• Gartland 분류 결정

• Type I 혹은 처음 골절이 의심되면 CT 촬영 고려

#### 고급 영상

• MRI: 혈관손상 혹은 신경손상 평가 필요 시

• CT: 회전 전위나 복잡한 골절선 평가

• Arteriography 혹은 Doppler: 혈관 손상 의심 시

(7) 치료

#### 비수술적 치료 (Conservative Treatment)

• Type I 골절

• 팔걸이 또는 부목으로 고정

• 2-3주간 고정

• 주기적 추적 평가

• Type II 골절

• 폐쇄 정복(Closed reduction)

• 동맥 혹은 연부조직 손상 없으면 조직 침투가 없을 때까지 대기 가능

• K-선에 의한 percutaneous pinning 추가 고려

• 무마취 정복 가능성 논란

#### 수술적 치료

• Type II (높은 재전위 위험) 및 Type III 골절

• Closed reduction + percutaneous pinning(K-선 고정)

• FPL(flexor pronator quad), FDP(2,3번), pronator quadratus를 통한 접근

• Pinning 기술

① 정복 후 K-선 2-3개로 고정

② 측면에서 내측 삽입(Medial approach)

• 척골신경 손상 위험

③ 양측 접근: 내측과 외측 모두에서 삽입

• 더 안정적

• 현재 추천 방법

• 개방 정복이 필요한 경우

① 폐쇄 정복 실패

② 신경 혹은 혈관 손상 중대

③ 골절선이 복잡함

#### 구획증후군(Compartment Syndrome)

• 응급 상황

• 통증, 부종, 창백, 감각마비, 마비 등 증상

• Fasciotomy 긴급 필요

• Anterior compartment(전방) 가장 흔함

(8) 합병증

#### 조기 합병증

• 신경 손상: 정중, 요골, 척골신경 손상

• 혈관 손상: 상완 동맥 손상으로 인한 손 허혈

• 구획증후군: 응급 fasciotomy 필요

• 피부 천공: 골편이 피부 관통

#### 후기 합병증

• 주각변형(Cubitus valgus): 외측 골편의 과도한 경사(varus tilting)로 인한 가장 중요한 합병증

• 주각내반(Cubitus varus): 내측 골편의 치유 문제

• 회전 변형(Rotational deformity)

• 원위부 상완골 성장 장애(Distal humerus growth disturbance)

• 골화 지연(Delayed union) 또는 불유합(Nonunion)

• Myositis ossificans: 드물지만 가능

• 팔꿈치 운동 범위 제한

• 척골신경 후발성 마비(Tardy ulnar nerve palsy)

(9) 예후

• 전체적으로 좋은 예후

• 대부분의 신경 손상은 자연 회복

• 적절한 정복과 고정으로 좋은 결과 기대

2) 상완골 과상 골절의 진단과 치료 원칙

(1) 초기 평가 순서

1. 임상 검진: 신경, 혈관 손상 평가

2. X선 촬영: Type 결정

3. 신경학적 기록: 기저선 자료

4. 혈관 평가: 맥박, 손의 색깔, 온도

(2) 신경손상 관리

• 즉시 신경학적 검진 기록

• 폐쇄 정복 후 재평가

• 신경 손상이 지속되면

• 신경 영상 검사(MRI, Electromyography) 고려

• 신경학 전문가 상담

• 조기 신경염전(Entrapment) 제거 고려

(3) 혈관손상 관리

• 즉시 대기 요법(Observation) 시작

• 폐쇄 정복 후 맥박과 손의 색깔 평가

• 개선되지 않으면

• 정복 재평가

• 혈관 영상 검사(Angiography, Doppler)

• 혈관 수술 상담

Kim's Solution

상완골 과상 골절의 진단 체크리스트:

1. FOOSH(Fall on outstretched hand) 외상력 확인

2. 팔꿈치 부종, 변형, 통증 평가

3. 신경 손상 여부: 정중신경(폄지, OK sign), 요골신경(배측지 신전), 척골신경(소지 외전) 평가

4. 혈관 손상 여부: 맥박, 손 색깔, 온도 평가

5. X선 촬영: Gartland 분류

6. Type I: 보존적 치료

7. Type II-III: 정복 필요 (폐쇄 또는 개방)

8. K-선 고정 고려

9. 신경/혈관 손상 주의

치료 결정 매트릭스:

- Type I: 팔걸이/부목 고정 2-3주

- Type II: 폐쇄 정복 + K-선 고정 고려 (안정성에 따라)

- Type III: 폐쇄 정복 + K-선 고정 또는 개방 정복 필요

- 신경손상 있어도: 폐쇄 정복 우선 시도, 개선 안 되면 신경 수술 고려

- 혈관손상: 대기 요법 가능하나 손의 색깔/맥박 악화 시 재평가 필수

Supracondylar 골절의 주각변형(Cubitus valgus) 예방:

- 외측 골편의 완전한 정복 필수

- 외측 골편의 varus tilting 최소화 (Lateral condyle overgrowth 예방)

- 해부학적 정복 실패 시 나중에 보정 어려움

- Type III 또는 회전 전위: 개방 정복 고려
