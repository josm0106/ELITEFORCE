제19단원

Epiphyseal plate

골단판

교과서 내용 정리

1) 골단판 (Epiphyseal plate)의 구조와 기능

(1) 골단판의 위치

• 골간부와 골단부 사이의 연골 판

• 골길이 성장 담당

• 미성숙 뼈에서 주요 성장 부위

(2) 골단의 성장률

#### (1) 상지 (upper limb)

• 근위 상완골: 40%

• 원위 상완골: 10%

• 근위 요골/척골: 10%

• 원위 요골/척골: 40%

• 척골골 분포에 비해 대칭 위치에 다양성 있음

#### (2) 하지 (lower limb)

• 근위 대퇴골: 15%

• 원위 대퇴골: 37%

• 근위 경골/비골: 28%

• 원위 경골/비골: 20%

• 비골은 골융합 비류에 더 영향 적음

2) 골단판의 조직학적 구조

(1) 조직학적 층 (Histologic zones)

#### (1) Resting zone (휴지대)

• 세포 분포가 중칙, 세포로 구성된 충격, 세포로 구성된 기질

#### (2) Proliferative zone (증식대)

• 빠른 세포 분열

• Type II collagen 함유

• Proliferative zone의 깊이가 성장 속도 반영

#### (3) Type IX collagen 함유 (MCD)

• 유기질의 주요 성분

• 연골 소주의 기질

#### (4) Hypertrophic zone (비후대)

• 세포 크기 증가

• 5-10배 크기의 세포

#### (5) Vascular invasion zone (혈관 침습대)

• 골화 시작 부위

• 2/3 정도는 골수로 인해지기, 1/3은 공간에서 기질 작용

3) 골단판 손상의 분류

(1) Salter-Harris 분류

• Type I: 골단판을 통한 골절

• Type II: 골단판과 골극부를 통한 골절

• Type III: 골단을 포함한 관절면 골절

• Type IV: 골극부와 골단 및 골간부를 통한 골절

• Type V: 골단판 자체의 압박 손상

(2) Peterson 분류 (Peterson classification)

• Type I: Epiphyseal separation (epiphyseal separation)

• Type II: Epiphyseal fracture with metaphyseal fragment

• Type III: Epiphyseal fracture (부분적 관절면 침범)

• Type IV: Complete epiphyseal fracture

• Type V: Complex fracture-dislocation

• Type VI: Loss of articular surface

4) 골단판 손상의 합병증

(1) 급성 합병증

• 혈관 손상

• 신경 손상

• 조직 손상

(2) 후기 합병증

• Bony bridge formation (골성교 형성)

• Angular deformity (각변형)

• Avascular necrosis (무혈성 괴사)

• Growth disturbance

(3) 특수 합병증

• Type III-IV 골절의 관절 손상

• Early osteoarthritis 가능성

• Functional impairment

5) 임상적 의의

(1) 진단

• 정확한 방사선 촬영 필요

• Salter-Harris 분류 결정

• MRI로 골단판 손상 정도 평가 가능

(2) 치료

• 정확한 정복

• 부정확한 정복 방지

• 정기적 추적 관찰

• 성장 장애 모니터링

(3) 예후

• Type I-II: 양호한 예후

• Type III-IV: 유합 및 변형 위험

• Type V: 성장 장애 가능성

Kim's Solution

골단판의 성장률

상지: prox. humerus (40%), distal humerus (10%), prox. radius/ulna (10%), distal radius/ulna (40%)

하지: prox. femur (15%), distal femur (37%), prox. tibia/fibia (28%), distal tibia/fibula (20%)

비골은 골융합에 비례하여 영향 적음

Hypertrophic zone의 중요성

1) 세포 크기 증가

2) Type X collagen 함유

3) Type X collagen의 변이: MCD (metalloproteinase collagenase)

4) 유기질이 기질 내 광상응에 더 양적 기질 부위

Vascular invasion zone

골화 시작 부위

충성인으로부터 충성 도관이 침투

2/3 정도는 골수로 인해지기, 1/3은 공간에서 기질 작용
