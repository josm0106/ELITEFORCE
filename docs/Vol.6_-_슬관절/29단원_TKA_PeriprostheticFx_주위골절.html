슬관절 (Knee Joint) Board Study Guide

29단원 — TKA Periprosthetic Fracture

2022 ELITE OS BOARD GUIDE Vol.6 기반 | 부민병원 정형외과

개요 (Introduction)

보철물 주위 골절(periprosthetic fracture)은 TKA 후 드문 합병증이지만 심각한 후유증을 초래할 수 있습니다.

  • 발생률: 0.3-1.0%
  • 위치: 대퇴골 상부, 경골, 슬개골

대퇴 원위부 주위 골절 (Supracondylar Fracture)

1. 위험 인자

  • 전방 대퇴 절흔(anterior femoral notching)
  • 골다공증
  • 류마티스 관절염
  • 스테로이드 사용
  • 여성
  • 재치환술
  • 신경학적 질환

2. Lewis-Rorabeck 분류

Type

특징

치료

Type I

비전위, 보철물 안정

보존적 (보조기)

Type II

전위, 보철물 안정

ORIF (역행 정맥내 정복 또는 금속판)

Type III

전위 또는 비전위, 보철물 이완

재치환술

3. 역행 정맥내 정복(Retrograde IMN) 금기

  • 폐쇄형 boxed design 보철물
  • 대퇴 보철물의 intercondylar distance < 못 직경
  • 줄기 연장 보철물

4. Type III 치료

  • 재치환술이 표준
  • 장 줄기(long stem) 사용
  • 필요시 골이식
  • 제약된(constrained) 보철물 고려

경골 주위 골절 (Tibial Periprosthetic Fracture)

1. 분류

  • 위치, 보철물 안정성, 결손의 정도에 따라 분류

2. 치료 원칙

  • 보철물 이완: 재치환술
  • 보철물 안정, 비전위: 보존적 치료
  • 보철물 안정, 전위: ORIF (측방 잠금 금속판 ± 내측 금속판)

3. 특수 상황

  • 근위 경골 다중편 골절: 다중 나사 및 잠금 나사
  • 경골 간부 골절: IMN (비줄기 보철물인 경우만 가능)

슬개골 주위 골절 (Patellar Periprosthetic Fracture)

1. 원인

  • 과도한 슬개골 절제
  • 측방 이완술 후 혈관 손상
  • 슬개골 추적 장애
  • 과도한 관절선 상승
  • 굴곡 >115°
  • 외상
  • PMMA 중합열로 인한 괴사

2. 분류 및 치료

Type

특징

치료

Type I

신전 기전 온전, 보철물 안정

비수술적 (보조기 6주)

Type II

신전 기전 단절, 전위

장력대 철선 + 지대 봉합

Type III

느슨한 보철물

보철물 제거 ± 부분 절제술

3. 특수 상황

  • 근위/원위극 골절: 부분 절제술 + 봉합
  • 극심한 분쇄: 전체 절제술 + 신전 기전 재건

전반적 고려사항

수술 시 예방

  • 과도한 전방 절흔 피하기
  • 적절한 크기의 보철물 선택
  • 복합면(PS) vs 후방지지(CR) 신중히 선택

✦ 연습문제 (5지선다)

문제 1. (2018) TKA 후 보철물 주위 골절의 가장 흔한 선행 요인으로 거리가 먼 것은?

① 전방 대퇴 절흔

② 류마티스 관절염

③ 골다공증

④ 스테로이드 사용

⑤ 골관절염

▶ 정답: ⑤ 골관절염

해설: 골관절염은 TKA의 가장 흔한 원인이지만 주위 골절의 특정한 선행 요인은 아닙니다.

문제 2. (2017) TKA 후 주위 골절의 선행 요인으로 옳은 것은?

① 골다공증

② 류마티스 관절염

③ 스테로이드 사용

④ 여성

⑤ 모두 옳다

▶ 정답: ⑤ 모두 옳다

해설: 모든 선택지가 TKA 후 주위 골절의 확립된 위험 인자입니다.

문제 3. (2016) 폐쇄형 box design 대퇴 보철물을 가진 환자의 대퇴 원위 전위 골절 시 최적의 치료는?

① 역행 정맥내 정복

② 보존적 치료

③ 측방 잠금 금속판 고정

④ 1단계 재치환술

⑤ 2단계 재치환술

▶ 정답: ③ 측방 잠금 금속판 고정

해설: 폐쇄형 box design은 역행 못 삽입이 불가능하므로 금속판 고정(ORIF)이 필요합니다.

문제 4. (2017) 22년 전 TKA를 받은 70세 환자가 낙상 후 대퇴 원위 전위 골절이 발생했습니다. 골다공증, 이완 없음. 최적 치료는?

① 폐쇄 정복 + 보조기

② 역행 정맥내 정복

③ 금속판 고정

④ 장 줄기 재치환술 + ORIF + 골이식

⑤ 단순 재치환술

▶ 정답: ③ 금속판 고정

해설: 보철물이 안정적이고 골 질이 양호하다면(골다공증이지만 이완 없음) ORIF가 표준입니다.

문제 5. (2016) TKA 후 슬개골 보철물 주위 골절 Type I의 치료는?

① 즉시 수술적 고정

② 보조기로 6주 비수술적 치료

③ 슬개 절제술

④ 관절 유합술

⑤ 재치환술

▶ 정답: ② 보조기로 6주 비수술적 치료

해설: 신전 기전이 온전하고 보철물이 안정적인 Type I은 비수술적 치료가 표준입니다.

문제 6. (2015) 경골 주위 다중편 골절에서 보철물이 안정적이고 골 질이 충분한 경우 가장 적절한 치료는?

① 보존적 치료

② 재치환술

③ 다중 이중피질 나사 + 잠금 나사와 측방 금속판

④ 외부 고정

⑤ 절제술

▶ 정답: ③ 다중 이중피질 나사 + 잠금 나사와 측방 금속판

해설: 안정적인 보철물과 충분한 근위 골편이 있으면 ORIF (다중 나사 + 측방판)이 표준입니다.

문제 7. (변형) TKA 시술 15년 후 넘어지면서 대퇴 원위 골절이 발생한 65세 여성 환자입니다. 전산화 단층촬영에서 보철물의 내측과 외측 확장부(condyle) 모두를 포함하는 골절선이 보이고, 보철물과의 해리는 없습니다. 치료 계획 수립 시 가장 중요한 고려사항은?

① 방사선학적 임상 해부학적 평가와 보철물 설계를 확인하여 역행 못 삽입의 가능성 평가

② 즉시 개방 정복 내고정을 시행하기 위한 수술 준비

③ 골다공증 치료가 우선이므로 비스포스포네이트 투여 후 수술

④ 흉부 영상 검사로 전신 마취 가능성 평가

⑤ 환자의 나이 때문에 재치환술 고려

▶ 정답: ① 방사선학적 임상 해부학적 평가와 보철물 설계를 확인하여 역행 못 삽입의 가능성 평가

해설: TKA 후 대퇴 원위 골절에서 치료 선택은 보철물의 안정성뿐 아니라 보철물 설계에 따라 결정됩니다. 폐쇄형 boxed design은 역행 못 삽입이 불가능하므로 미리 확인이 필수입니다. ②는 보철물 안정성이 확인되었다면 ORIF가 가능하므로 즉시 수술만이 정답이 아니고, ③은 수술 준비 전에 필요한 초기 검사가 아니며, ④는 수술 적응증과 무관하고, ⑤는 보철물이 안정적이므로 재치환이 필수가 아닙니다.

문제 8. (변형) TKA 후 경골 근위부 주변에서 다중편 골절이 발생한 환자에서 보철물이 안정적일 때, 골절편 고정 시 주의할 점으로 틀린 것은?

① 나사는 보철물 거리보다 더 짧은 것을 선택

② 여러 개의 이중피질 나사와 잠금 나사를 사용

③ 측방 금속판을 보강재로 추가

④ 각 골절편에 최소 2개의 나사로 고정

⑤ 내측 금속판을 우선적으로 적용

▶ 정답: ⑤ 내측 금속판을 우선적으로 적용

해설: 경골 근위부 다중편 골절에서는 측방 금속판이 주요 지지 장치로 작용하므로, 우선 측방 금속판으로 기본 지지를 제공하고 필요시 내측판을 보조로 사용합니다. ①은 보철물 손상 회피의 기본 원칙이고, ②③④는 모두 다중편 골절의 안정화를 위해 필수적입니다. 내측판이 우선되면 보철물과의 간섭이 증가될 수 있습니다.

문제 9. (변형) TKA 시술 20년 후 슬개골 보철물 주위에서 골절이 발생했습니다. 방사선 검사에서 Type III 골절로 진단되었습니다. 이 경우 최적의 치료 방향으로 옳은 것은?

① 보조기 고정만으로 충분

② 전위의 정도와 보철물 이완 상태를 구분하여 치료 결정

③ 모든 Type III 골절에서 슬개골절제술이 필수

④ 보철물이 있으므로 금속 고정만 가능

⑤ 즉시 재치환술

▶ 정답: ② 전위의 정도와 보철물 이완 상태를 구분하여 치료 결정

해설: Type III 슬개골 보철물 주위 골절은 보철물 이완 여부에 따라 치료가 다릅니다. 보철물이 안정적이면서 전위가 최소하면 보존적 치료가 가능하지만, 이완이 있거나 전위가 심하면 재치환술이 필요합니다. ①은 전위가 있으면 부적절하고, ③은 과도한 치료이며, ④는 재치환술이 불가능한 상황을 가정한 것이고, ⑤는 모든 경우에 해당되지 않습니다.

문제 10. (변형) TKA 6개월 후 낙상으로 인한 근위 경골 골절 환자입니다. 골절은 비전위이고 보철물이 안정적입니다. 수술적 고정 시 역행 수내 정복의 가능성을 평가할 때 가장 중요한 기준은?

① 환자의 나이와 골질의 상태

② 골절편의 개수와 골손상의 정도

③ 보철물이 줄기 연장 보철물이 아닌 비줄기형인지 여부

④ X선 촬영의 회전 정도

⑤ 수술 시간과 마취 시간의 제약

▶ 정답: ③ 보철물이 줄기 연장 보철물이 아닌 비줄기형인지 여부

해설: TKA 후 경골 골절에서 역행 정맥내 정복이 가능하려면 보철물이 비줄기형(non-stemmed design)이어야 합니다. 줄기 연장 보철물은 수내정이 물리적으로 불가능합니다. ①은 보존적 치료 여부에는 영향을 주지만 역행 정맥내 정복 적응증에는 아니고, ②③④는 고정 방법의 기술적 선택에 관련되며, ⑤는 치료 선택과 무관합니다.
