제19단원

샤르코관절병증 (Charcot Arthropathy)

1. 특성

• 신경병증이 동반된 비감염성 골 파괴

2. 원인

• DM neuropathy, Hansen neuropathy, syringomyelia

3. 증상

• Swelling, painless, instability - 3대 증상

4. 분류 (Brodsky)

• Type I: 족근-중족 관절 (60%)

• Type II: Chopart & subtalar 관절 (25%)

• Type III: 발목 관절 (10%)

5. Eichenholtz 병기

• 1단계: 급성, 부종, 불안정성

• 2단계: 활동기, 골편 흡수, 새 골 형성

• 3단계: 만성, 부종 소실, 정렬이상

6. 치료

• 1단계: Total contact cast, 고정

• 2-3단계: 부팅 또는 맞춤형 정형용구

• 수술: 불안정성, 궤양 시 고려

7. 기타 치료

• Bisphosphonate, Calcitonin: 제한적 효과

• Vitamin D 보충
