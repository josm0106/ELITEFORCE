거골 경부/체부 골절 (Talar Neck & Body Fracture)

1. 해부학적 배경 (Anatomical Background)

• 거골의 혈액 공급: 주로 측면에서 진입

• 경부 골절 시 혈관 손상 위험 높음

• 무혈성 괴사 (AVN) 위험 증가

2. 거골 경부 골절 (Talar Neck Fracture)

2-1. 분류 (Hawkins Classification)

2-2. 치료

• Type I: 보존적 치료 (6-8주 고정)

• Type II, III, IV: ORIF 필수

• 조기 정복 필수 (24시간 내)

3. 거골 체부 골절 (Talar Body Fracture)

3-1. 임상양상

• 심각한 종골 및 경골 손상과 동반 가능

• 심각한 연부조직 손상

• 높은 치명률과 이환율

3-2. 치료

• ORIF 또는 관절유합술

• 필요시 다단계 수술

• 후방 접근법 필요하기도 함

4. 합병증 (Complications)

4-1. 무혈성 괴사 (AVN)

• Type II: 20-50%

• Type III: 50-100%

• Type IV: 거의 100%

4-2. 기타 합병증

• Post-traumatic arthritis

• Nonunion

• 신경 손상 (깊은 비골신경)

• 감염

5. 예후 (Prognosis)

• Type I: 우수

• Type II, III: 중등도

• Type IV: 불량

단답형 문제

1. 거골의 혈액 공급의 특징은?

답: 주로 측면에서 진입하므로 경부 골절 시 혈관 손상 위험이 높습니다.

2. Hawkins classification Type IV의 특징은?

답: 거골상관절과 거골하관절이 모두 탈구된 형태로 거의 100%의 AVN 위험이 있습니다.

3. Type I과 Type II, III, IV의 치료 방침 차이는?

답: Type I: 보존적 치료, Type II-IV: ORIF 필수

4. 거골 경부 골절에서 AVN의 위험 요소는?

답: 골절 전위 정도, 혈관 손상, 정복 지연, 감염

5. 거골 체부 골절의 특징은?

답: 심각한 다중 손상과 연부조직 손상을 동반하며 높은 이환율을 보입니다.

5지선다형 문제

1. 거골 경부 골절의 Hawkins Type I의 치료 원칙은?

① 즉시 수술

② 보존적 치료 (6-8주 고정)

③ 관절유합술

④ 방치

⑤ 항생제만

정답: ②

해설: Type I은 비전위 골절로 보존적 치료가 가능합니다.

2. Hawkins Type III의 AVN$ 위험도는?

① 0-10%

② 10-20%

③ 20-50%

④ 50-100%

⑤ 100%

정답: ④

해설: Type III는 50-100%의 높은 AVN$ 위험이 있습니다.

3. 거골 경부 골절에서 정복 지연의 주요 위험은?

① 부정유합만

② 무혈성 괴사 위험 증가

③ 감염만

④ 신경 손상만

⑤ 출혈만

정답: ②

해설: 조기 정복은 AVN 위험을 감소시킵니다.

4. 거골 체부 골절의 가장 특징적인 임상 양상은?

① 경미한 손상

② 심각한 다중 손상과 연부조직 손상

③ 단순 통증

④ 부종만

⑤ 무증상

정답: ②

해설: 거골 체부 골절은 심각한 다중 손상을 동반합니다.

5. Hawkins Type IV의 AVN 위험도는?

① 20%

② 50%

③ 거의 100%

④ 0%

⑤ 50% 이하

정답: ③

해설: Type IV는 거의 100%의 AVN$ 위험을 보입니다.

6. 거골 경부 골절 환자에서 조기 정복 시기는?

① 1주일 후

② 24시간 내

③ 2-3주 후

④ 1개월 후

⑤ 필요 없음

정답: ②

해설: 조기 정복은 24시간 내에 시행하는 것이 권장됩니다.

7. 거골 골절의 가장 심각한 합병증은?

① 부종만

② 무혈성 괴사 (AVN)

③ 감염만

④ 신경 손상만

⑤ 혈종만

정답: ②

해설: AVN은 장기 기능성을 심각하게 손상시킵니다.

8. 거골 경부 골절에서 정복 확인의 가장 좋은 방법은?

① 일반 X-ray만

② CT$ 스캔

③ MRI

④ 초음파

⑤ 진찰만

정답: ②

해설: CT$ 스캔으로 정복 정도와 관절면 정렬을 정확히 평가할 수 있습니다.
