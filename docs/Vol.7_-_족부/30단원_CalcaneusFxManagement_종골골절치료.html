종골골절 치료 (Calcaneus Fracture Management)

1. 보존적 치료 (Conservative Treatment)

• Sanders Type I, II: 보존적 치료 가능

• 얼음, 거상(elevation), 압박(compression)

• 4-6주 고정 후 물리치료 시작

2. 수술적 치료 (Operative Treatment)

2-1. 적응증

• Sanders Type III, IV: 수술 고려

• 전위 골절 (displaced fractures)

• Bohler angle > 20° 감소

2-2. 수술 기법

• ORIF (Open Reduction Internal Fixation)

• Percutaneous screw fixation - 최소침습

• Lateral plate fixation

• Primary fusion - 심한 손상 시

3. 수술 후 관리 (Postoperative Management)

• 6-8주 체중 부하 금지

• 물리치료 및 점진적 재활

• 운동 범위 회복 프로그램

4. 특수 상황 (Special Situations)

• 폐쇄 정복 실패 시 수술 전환

• 연부조직 손상 심할 때 다단계 수술

단답형 문제

1. Sanders Type I, II의 치료 원칙은?

답: 보존적 치료: 얼음, 거상, 압박, 4-6주 고정 후 물리치료

2. Sanders Type III, IV의 치료 방침은?

답: 수술적 치료 고려: ORIF 또는 percutaneous fixation

3. 종골골절 수술 후 체중 부하 시기는?

답: 6-8주 체중 부하 금지 후 점진적 부하

4. 최소침습 수술의 방법은?

답: Percutaneous screw fixation

5. 종골골절 수술 후 주요 재활 과정은?

답: 물리치료, 운동 범위 회복 프로그램, 점진적 부하

5지선다형 문제

1. Sanders Type I, II의 가장 적절한 초기 치료는?

① 즉시 수술

② 보존적 치료 (얼음, 거상, 압박)

③ 무조건 운동

④ 절대 부하

⑤ 관찰만

정답: ②

해설: Sanders Type I, II는 비전위 또는 부분 분쇄 골절로 보존적 치료가 효과적입니다.

2. 종골골절 수술적 치료의 주요 목표는?

① 비용 절감

② Bohler angle 복구 및 관절면 복원

③ 빠른 수술

④ 최소 절개만

⑤ 감염 방지만

정답: ②

해설: Bohler angle의 복구와 관절면 정렬이 장기 예후를 좌우합니다.

3. Percutaneous screw fixation의 장점은?

① 매우 복잡

② 최소침습, 연부조직 손상 최소화

③ 항상 최고의 방법

④ 보존적 치료만 가능

⑤ 높은 감염률

정답: ②

해설: Percutaneous fixation은 연부조직 손상을 최소화하는 장점이 있습니다.

4. 종골골절 수술 후 체중 부하를 금지하는 이유는?

① 별 이유 없음

② 골절 치유 시간 확보

③ 환자 편의

④ 감염 방지

⑤ 신경 보호

정답: ②

해설: 충분한 골 유합을 위해 6-8주 체중 부하를 피해야 합니다.

5. 종골골절 치료에서 ORIF의 적응증은?

① 모든 비전위 골절

② Sanders Type III, IV와 전위 골절

③ Type I만

④ Type II만

⑤ 나이가 많을 때만

정답: ②

해설: ORIF는 분쇄가 심한 경우에 적응됩니다.
