Part 2. 성장판 (Epiphyseal Plate / Growth Plate)

1) 성장판 구역 분류

골단(Epiphysis) 쪽에서 골간단(Metaphysis) 쪽으로:

정지대 → 증식대 → 성숙대 → 비후대 → 퇴행대(혈관 침범대)

★ 외력에 가장 취약한 부위: 비후대 (Hypertrophic zone)

- SCFE (Slipped Capital Femoral Epiphysis)도 비후대에서 발생

2) 정지대 (Reserve Zone)

- 골단에 가장 가까운 부분

- 풍부한 기질과 미성숙한 연골 세포로 구성

- Type II collagen의 함량이 가장 높다

- 드물게 세포 분열 (길이성장에 관여하지 않음)

- 세포 내 칼슘 및 이온화된 칼슘 함량이 가장 적다

3) 증식대 (Proliferative Zone) ★★★

- 연골세포의 세포 분열이 왕성

- 납작하고 편평해진 세포들이 동전을 쌓아 놓은 것처럼 기둥 모양으로 배열

- 길이성장에 관여

- 산소 분압이 가장 높아서 단백다당의 합성과 전환이 가장 활발

- 방사선 조사에 가장 영향을 많이 받는 부위

4) 성숙대 (Maturation Zone)

- 세포외 기질의 합성이 이루어지고 세포 분열이 완성

- Type II collagen 돌연변이 → SED, Kniest dysplasia, Stickler syndrome

- Type IX collagen, COMP 돌연변이 → MED, Pseudochondroplasia

- Type XI collagen 돌연변이 → Stickler syndrome

5) 비후대 (Hypertrophic Zone) ★★

- 세포 분열 중료, 세포의 크기는 증식대의 약 5-10배

- Type X collagen 합성

- Collagen X는 오직 관절 연골의 석회화 연골층 (Zone of calcified cartilage)과 성장판의 비후대에서만 발견

- → 연골의 석회화에 역할을 하는 것으로 추정

- Type X collagen 변이 = MCD (다발성 연골이형성증)

- 무기질(칼슘) 침착 작용이 나타나며 hydroxyapatite로 침착

- 산소 분압이 가장 낮고, 혈관이 없는 구조로 외력에 가장 약한 부위

6) 혈관 침범대 (Vascular Invasion Zone)

- 골간단으로부터 들어온 모세혈관이 비후대에 있는 무기질이 침착된 연골로 침범

- 2/3 정도는 흡수되어 없어지고, 1/3은 골모세포에 의해 골 기질 침착

관련 문제 정리 (서술형)

Q1. 골수에서 분리한 미분화 줄기세포를 연골화 분화시켜 관절연골 재생에 사용할 때, Collagen X를 가지고 있는 연골의 단점은?

→ 정답: 석회화 (Calcification)

해설: Type X collagen은 석회화 연골층과 성장판 비후대에서만 발견되며, 연골의 석회화에 관여하는 것으로 추정된다.

Q2. 골단판에서 연골세포의 세포 분열이 왕성하고, 길이성장에 관여하며, 방사선 조사에 가장 영향을 많이 받는 부위는?

→ 정답: 증식대 (Proliferative Zone)

해설: 산소 분압이 가장 높아서 단백다당의 합성과 전환이 가장 활발. 방사선에 가장 민감한 부위.

Q3. 관절연골(articular cartilage)의 deep zone과 성장판(physis)의 maturation zone에 공통적으로 가장 많이 관찰되는 교원질은?

→ 정답: Type II collagen

해설: 관절 연골과 성장판 모두 주된 교원질은 Type II collagen이다.

Q4. 박리성 골연골염에 대해 미세 골절술 후 초자연골이 아닌 섬유연골로 재생될 때 감소하는 연골 기질 주성분은?

→ 정답: Type II collagen

해설: 초자연골(hyaline cartilage) = Type II, 섬유연골(fibrous cartilage) = Type I. 섬유연골로 전환 시 Type II 감소.

Q5. 골조직, 관절 연골, 건(tendon)을 구성하는 주된 교원질의 형은?

→ 정답: 골조직: Type I, 관절 연골: Type II, 건: Type I

해설: Type I = 단단한 조직(골, 건, 섬유연골), Type II = 물렁한 조직(초자연골, 수핵).

Q6. 성장판에서 Zone A(성숙대 상부)와 Zone B(비후대)에서 주로 발현되는 콜라겐형은?

→ 정답: Zone A: Type II, Zone B: Type X

해설: 비후대에서만 Type X collagen이 합성된다. 성숙대까지는 Type II가 주된 교원질.

연습문제 (5지선다 객관식)

1. 다음 중 조직과 교원질의 연결이 옳지 않은 것은?

① Hyaline cartilage — Type II collagen

② Bone — Type I collagen

③ Nucleus pulposus — Type I collagen

④ Tendon — Type I collagen

⑤ Osteogenesis imperfecta — Type I collagen

정답: ③

해설: Nucleus pulposus(수핵)의 교원질은 Type II collagen이다. Type I이 아니다. 수핵은 초자연골과 유사한 '물렁한' 조직이므로 Type II이다.

2. 성장판의 증식대 (Proliferative Zone)에 대한 설명으로 옳지 않은 것은?

① 연골세포의 세포 분열이 왕성하다

② 납작하고 편평해진 세포들이 기둥 모양으로 배열된다

③ 산소 분압이 가장 높아서 단백다당의 합성이 가장 활발하다

④ 방사선 조사에 가장 영향을 많이 받는 부위이다

⑤ Type X collagen이 합성되는 부위이다

정답: ⑤

해설: Type X collagen은 증식대가 아닌 비후대 (Hypertrophic Zone)에서 합성된다. 증식대의 특징은 O₂↑, 분열↑, 방사선 민감이다.

3. 성장판의 비후대 (Hypertrophic Zone)에 대한 설명으로 옳지 않은 것은?

① Type X collagen이 합성된다

② 산소 분압이 가장 낮다

③ 세포의 크기는 증식대의 약 5-10배이다

④ 외력에 가장 강한 부위이다

⑤ 무기질 침착이 나타나며 hydroxyapatite로 침착된다

정답: ④

해설: 비후대는 외력에 가장 약한 부위이다. '강한'이 아니라 '약한'이 정답. SCFE도 비후대에서 발생한다.

4. 정지대 (Reserve Zone)에 대한 설명으로 옳지 않은 것은?

① 골단에 가장 가까운 부분이다

② Type II collagen의 함량이 가장 높다

③ 세포 내 칼슘 및 이온화된 칼슘 함량이 가장 많다

④ 풍부한 기질과 미성숙한 연골 세포로 구성된다

⑤ 드물게 세포 분열을 한다

정답: ③

해설: 정지대는 세포 내 칼슘 및 이온화된 칼슘 함량이 가장 적다. '가장 많다'가 아니다.

5. Collagen X에 대한 설명으로 옳지 않은 것은?

① 성장판의 비후대에서 합성된다

② 관절 연골의 석회화 연골층에서도 발견된다

③ 연골의 석회화에 역할을 하는 것으로 추정된다

④ 돌연변이 시 MED (Multiple Epiphyseal Dysplasia)를 유발한다

⑤ 돌연변이 시 MCD (Multiple Chondrodysplasia)를 유발한다

정답: ④

해설: Type X collagen 변이는 MCD를 유발한다. MED는 Type IX collagen과 COMP의 돌연변이에 의해 발생한다.

6. 다음 중 성장판 구역에 대한 설명으로 올바른 것은?

① 정지대는 세포 내 칼슘 함량이 가장 많다

② 증식대는 납작하고 편평해진 세포들이 기둥 모양으로 배열되어 있으며 방사선 조사에 영향을 받는 부위이다

③ 성숙대는 산소 분압이 가장 높아서 단백다당의 합성이 가장 활발하다

④ 비후대는 산소 분압이 가장 낮고, 혈관이 없는 구조로 외력에 가장 강한 부위이다

⑤ 정지대에 있는 성숙한 연골 세포는 활발하게 세포 분열을 한다

정답: ②

해설: ②가 정답. 정지대=칼슘 최소(①X), 성숙대≠O₂ 최고(③X, 증식대가 O₂ 최고), 비후대=외력 최약(④X), 정지대=분열 드묾(⑤X).

7. 골수에서 분리한 미분화 줄기세포를 연골화 분화시킬 때, 세포 수준에서 분화 여부를 판단하기 위한 가장 적절한 방법은?

① Type I 교원질 합성량

② II형 교원질 합성량

③ I형 교원질 합성량

④ X형 교원질 합성량

⑤ 메탈로프로테이나제(metalloproteinase)활성도

정답: ②

해설: 관절연골의 주된 교원질은 Type II collagen이므로, 연골화 분화 시 II형 교원질 합성량을 측정하는 것이 가장 적절하다.

Take-Home Message

▸ Type I collagen = 골, 건, 섬유연골, 섬유륜, OI. 'I = 단단한(rigid)' 조직.

▸ Type II collagen = 초자연골, 수핵, 연골종판, SED. 'II = 물렁한(soft)' 조직.

▸ Type X collagen = 석회화 연골층 + 성장판 비후대에서만 발견. 변이 = MCD.

▸ 성장판 증식대: O₂ 최고, 단백다당 합성 최활발, 방사선 가장 민감, 기둥형 배열, 길이성장 관여.

▸ 성장판 비후대: O₂ 최저, 혈관 무, 외력에 가장 약함, SCFE 발생부위, Type X 합성, 세포 5-10배.

▸ 정지대: Type II collagen 최다, 칼슘 최소, 분열 드묾, 골단에 가장 가까움.

▸ Microfracture 후 초자연골→섬유연골 전환 시 Type II collagen 감소 (Type I이 대체).

▸ ★ 시험 함정: 수핵=Type I(X)/Type II(O). 비후대=외력 강함(X)/약함(O). 정지대=칼슘 많음(X)/적음(O).
